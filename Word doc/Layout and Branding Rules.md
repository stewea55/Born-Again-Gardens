Text between asterisks *like this* is bolded.
Text between two underscores _like this_ is italicised. 
\ denotes a new line/line break.

< > denotes a clickable link within the copy; don't actually use the "<"or ">" in the copy
<< >> denotes a button within the copy; don’t actually use the “<<” or “>>” in the copy

[ ] tells you about the link, button, or copy. The text between the brackets is not part of the copy, unless it is “[...]” in which case, the entire thing is part of the copy.
